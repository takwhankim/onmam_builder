# Onmam Builder

온맘닷컴 교회 홈페이지 제작/납품용 빌더 프로토타입입니다.

## 개요

이 프로젝트는 온맘닷컴 제작자가 교회 홈페이지를 빠르게 구성하고, 완성된 홈페이지를 교회에 납품하기 위한 단일 HTML 기반 빌더입니다.

현재 버전은 프론트엔드 프로토타입이며, 데이터 저장은 브라우저 로컬 저장소 중심으로 동작합니다. 추후 서비스 운영을 위해 백엔드, DB, 파일 업로드, 권한 관리, 교회별 관리자 화면을 분리해 붙일 예정입니다.

## 현재 파일

- `온맘-교회-홈페이지-빌더_9.html`

브라우저에서 이 파일을 직접 열면 빌더를 사용할 수 있습니다.

## 주요 기능

- 온맘 제작자 모드
- 교회 운영 관리 모드
- 방문자 화면 미리보기
- 템플릿 기반 홈페이지 생성
- 섹션 추가/편집/순서 변경
- 메뉴 및 하위 메뉴 관리
- 페이지/게시판 연결
- 자유게시판, 자료실, 사진 게시판, 영상 게시판
- Tiptap 기반 편집기 로딩 시도 및 내장 편집기 fallback
- 로고 이미지 설정
- 카테고리/서브메뉴 화면 구성
- 모바일/태블릿 미리보기
- 납품용 HTML 내보내기
